# hermes-food

## Summary

* [Instalation](#Instalation)

### Instalation

1. Rename the file `.env.example` to `.env` and setup your environment variables;
2. Run the command below:
```bash
$ make run-build
```
3. Run the database migration with this command:
```bash
$ make migration
```

### Tests:

Execute the command below:
```bash
$ make tests
```
